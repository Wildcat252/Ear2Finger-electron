1) Overview

Import a YouTube URL → Extract subtitles → Segment into timestamped sentences → Practice with per-word inputs and keyboard shortcuts.

  • YouTube Import + Subtitle Processing
  • Dictation workspace with per-word typing
  • Session Progress Tracking (Correct/Hint/Incorrect)
  • AI Coach Feedback and Practice Recommendations

The AI coach uses your practice history plus vector search to surface sentences based on your current weak spots.

• Install from source code

You can run Ear2Finger locally for development from the source code:

Backend

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn main:app --reload --host 0.0.0.0 --port 8000

Frontend

cd frontend
npm install
npm run dev

3) How to use

  • Open the app in your browser (default: http://localhost:3000).
  • Import a YouTube lesson (will extract subtitles, download audio, and build a dictation session).
  • Practice sentence-by-sentence:
    • Use keyboard shortcuts to move between words / sentences
    • Type what you hear; get underline guidance, hints, and looped playback
  • When you’re ready, open Ask coach to get AI feedback and recommendations.

5) FAQs

Do I need an AI key?

Yes, to use the AI coach features. Configure your provider keys in the app Settings.

What does “Import” do?

It downloads subtitles for the selected YouTube URL, segments them into sentences, and stores timestamps + practice history for dictation.

Is my practice data stored locally?

By default, this project stores data locally (SQLite + Qdrant). For cloud deployments, it depends on where you host the database and vector store.
